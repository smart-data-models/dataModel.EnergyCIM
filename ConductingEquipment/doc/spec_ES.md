## Lista de propiedades  

- `BaseVoltage`: Todos los equipos conductores con esta tensión base.  Utilizar sólo cuando no se utiliza ningún contenedor de nivel de tensión y sólo se aplica una tensión base.  Por ejemplo, no se utiliza para los transformadores. Por defecto: Ninguno  - `SvStatus`: La variable de estado asociada a este equipo conductor. Por defecto: Ninguno  - `Terminals`: Los equipos conductores tienen terminales que pueden estar conectados a otros terminales de equipos conductores a través de nodos de conectividad o nodos topológicos.
