Entity: ExcST2A  
===============  
[Open License](https://github.com/smart-data-models//dataModel.EnergyCIM/blob/master/ExcST2A/LICENSE.md)  
Global description: **Adapted from CIM data models. Modified IEEE ST2A static excitation system - another lead-lag block added to match  the model defined by WECC.**
