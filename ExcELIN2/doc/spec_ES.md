Descripción global: **Adaptado de los modelos de datos CIM. Modelo detallado del sistema de excitación - ELIN (VATECH).  Este modelo representa un sistema de excitación totalmente estático. Un controlador de tensión PI establece un punto de ajuste de corriente de campo deseado para un controlador de corriente proporcional. El integrador del controlador PI tiene una entrada de seguimiento para ajustar su señal a la corriente de campo actual.  Los modelos de estabilizadores del sistema de potencia utilizados junto con este modelo de sistema de excitación: PssELIN2, PssIEEE2B, Pss2B.**
