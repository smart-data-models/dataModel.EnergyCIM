Globale Beschreibung: **Abgeleitet aus CIM-Datenmodellen. Ein Leiter oder eine Gruppe von Leitern mit vernachlässigbarer Impedanz, die dazu dienen, andere leitende Geräte innerhalb einer einzelnen Unterstation zu verbinden.  Spannungsmessungen werden typischerweise von Spannungswandlern gewonnen, die an Sammelschienenabschnitte angeschlossen sind. Ein Sammelschienenabschnitt kann viele physikalische Anschlüsse haben, wird aber für die Analyse mit genau einem logischen Anschluss modelliert.**
