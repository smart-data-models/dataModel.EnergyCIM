Description globale : **Adapté des modèles de données CIM. La classe représente le modèle IEEE Std 421.5-2005 de type AC1A. Le modèle représente les systèmes d'excitation d'alternateur-rectificateur commandés par le champ désignés par le type AC1A. Ces systèmes d'excitation se composent d'une excitatrice principale d'alternateur avec des redresseurs non contrôlés.  Référence : IEEE Standard 421.5-2005 Section 6.1.**
