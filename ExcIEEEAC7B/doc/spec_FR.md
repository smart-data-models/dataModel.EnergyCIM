Entité : ExcIEEEAC7B  
====================  
[Licence ouverte] (https://github.com/smart-data-models//dataModel.EnergyCIM/blob/master/ExcIEEEAC7B/LICENSE.md)  
Description globale : **Adapté des modèles de données CIM. La classe représente le modèle IEEE Std 421.5-2005 type AC7B. Le modèle représente les systèmes d'excitation qui consistent en un alternateur c.a. avec des redresseurs fixes ou rotatifs pour produire le champ c.c. requis. Il s'agit d'une mise à niveau des systèmes d'excitation à courant alternatif antérieurs, qui ne remplacent que les commandes mais conservent l'alternateur à courant alternatif et le pont redresseur à diodes.  Référence : IEEE Standard 421.5-2005 Section 6.7.  Dans la norme IEEE 421.5 - 2005, le bloc [1 / sT] est représenté par [1 / (1 + sT)], ce qui est incorrect**.

Ce modèle de données est une conversion directe du modèle d'information commun (CIM) spécifié par la norme IEC61970 en modèles de données intelligents. Les classes python sur lesquelles ce modèle est basé ont été développées par ces entités : Institute for Automation of Complex Power Systems (ACS), EON Energy Research Center (EONERC) et RWTH University Aachen, Allemagne.
